# CrSaSo — índice de entidades (NotebookLM)

Documento en **Markdown convencional** para subir a NotebookLM o usar como contexto del manuscrito.

Cada entrada incluye:

- **Nombre:** enlace al archivo fuente `.mdx` en el repositorio
- **URL del códice:** ruta pública en el sitio Docusaurus
- **Categoría:** carpeta bajo `docs/CrSaSo/`

Al redactar fichas para el sitio, el equivalente en MDX es:
`import Link from '@docusaurus/Link'` y `<Link to="URL del códice">texto</Link>` (ver `_links-crSaSo.md`).
